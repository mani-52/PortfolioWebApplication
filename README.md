# Portfolio Web Application

A simple portfolio web application built with Flask, a lightweight Python web framework. This project showcases a personal portfolio with pages for Home, About, Projects, and Contact, styled with basic CSS. It was developed as a learning project in PyCharm and is deployed locally for demonstration purposes.

## Features

- **Homepage**: A welcoming introduction to the portfolio.
- **About Page**: Information about the developer, including skills and background.
- **Projects Page**: A list of projects with descriptions.
- **Contact Page**: Contact details for reaching out.
- **Responsive Design**: Basic CSS styling with a navigation bar and centered content layout.

## Prerequisites

Before you begin, ensure you have the following installed:

- Python 3.8 or higher
- pip (Python package manager)
- Git (for cloning the repository)
- A code editor like PyCharm (optional but recommended)
